# Installation  

### Clone the repository
```
git clone https://github.com/hmes98318/MusicBot.git
```

### Revise `config.json`
```json
{
    "token": "yourToken",
    "prefix": "+",
    "playing": "+help | music",
    "maxVol": 200
}
```

## Running the script 

```
node index.js
```

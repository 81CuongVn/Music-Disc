# Discord Music Bot
 
still in development  
### Reference version  
**node.js  `v16.13.2`**  
**discord.js  `v13.6.0`** 



# Installation

### Clone the repository
```
git clone https://github.com/hmes98318/MusicBot.git
```

### Install the depedences
```
npm install discord.js discord-player ffmpeg-static ms opusscript express    
```
